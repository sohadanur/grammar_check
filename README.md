# grammar_check
Grammar check using Gemini's model: gemini pro and fastapi. 
Worked on vscode.
To get gemini_api_key: https://www.youtube.com/watch?v=ywPDnh4T2TA&t=1s 
JSOn format to be written in the post request: 
{
    "sentences":[
         {"line": "This is apples"},
         {"line": "birds flies over sky!"}
    ]
}
